[Kudu] 4. мқёмҪ”л”©кіј м••м¶•

Kuduм—җм„ңлҠ” м»¬лҹјмқҳ мқёмҪ”л”©кіј м••м¶•мқ„ нҶөн•ҙ м ҖмһҘ лҚ°мқҙн„°мқҳ мӮ¬мқҙмҰҲлҘј мӨ„м—¬, л””мҠӨнҒ¬ кіөк°„мқ„ нҡЁмңЁм ҒмңјлЎң мӮ¬мҡ©н•  мҲҳ мһҲлҸ„лЎқ н•ҳкі  мһҲлӢӨ.

1. мқёмҪ”л”©

мқёмҪ”л”©мқҖ н…Ңмқҙлё” мғқм„ұ мӢң м»¬лҹјмқҳ м •мқҳлЎң м§Җм •н•  мҲҳ мһҲлӢӨ.

  id BIGINT PRIMARY KEY,
  c1 BIGINT ENCODING PLAIN_ENCODING,
  c2 BIGINT ENCODING AUTO_ENCODING,

1) Plain encoding

мқёмҪ”л”©мқ„ н•ҳм§Җ м•Ҡкі  мӣҗліё лҚ°мқҙн„° к·ёлҢҖлЎң м ҖмһҘн•ңлӢӨ.

2) Bitshuffle encoding

л№„нҠёлҘј мһ¬л°°м№ҳ н•ң л’Ө LZ4 л°©мӢқмңјлЎң м••м¶•н•ңлӢӨ. primary key лЎң м •л ¬н–Ҳмқ„л•Ң к°’мқҙ л°ҳліөлҗҳкұ°лӮҳ, мҲңм°Ём ҒмңјлЎң мҰқк°Җ/к°җмҶҢ н•ҳкұ°лӮҳ, мқјл¶Җмқҳ к°’л§Ң ліҖкІҪмқҙ лҗ л•Ң нҡЁмңЁм ҒмқҙлӢӨ.

м»¬лҹјмқҳ к°’л“Өмқ„ л°”мқҙл„ҲлҰ¬мҠӨнҠёлҰј нҳ•нғңлЎң ліҖкІҪ н•ң нӣ„ LZ4лЎң м••м¶•н•ңлӢӨ.

3) Run Length encoding

primary keyлЎң м •л ¬ н–Ҳмқ„л•Ң л°ҳліөлҗҳлҠ” лҸҷмқјн•ң к°’мқ„ 1к°ңмқҳ к°’кіј л°ҳліө нҡҹмҲҳл§Ң м ҖмһҘн•ҳм—¬ м ҖмһҘнҒ¬кё°лҘј мӨ„мқҙлҠ” мқёмҪ”л”©мқҙлӢӨ. лҸҷмқјн•ң к°’мқҙ м—°мҶҚм ҒмңјлЎң м ҖмһҘлҗ  мҲҳлЎқ нҡЁмңЁм ҒмқҙлӢӨ.

4) Dictionary encoding

м»¬лҹјмқҳ к°’мңјлЎң Directionary к°Җ мғқм„ұлҗҳкі , мӢӨм ң м»¬лҹјмқҳ к°’м—җлҠ” н•ҙлӢ№ Dictionaryмқҳ мқёлҚұмҠӨк°Җ м ҖмһҘлҗңлӢӨ. Dictionary EncodingмқҖ лҸҷмқјн•ң к°’мқҙ л°ҳліөлҗҳл©°, мң лӢҲнҒ¬н•ң к°’мқҙ м ҒмқҖ кІҪмҡ° нҡЁмңЁм ҒмқҙлӢӨ.

5) Prefix encoding

кіөнҶөлҗң prefixлҘј к°–лҠ” кІҪмҡ°, ліҖкІҪлҗҳлҠ” л¶Җл¶„л§Ң м ҖмһҘлҗҳлҠ” л°©мӢқмқҙлӢӨ. мҳҲлҘј л“Өм–ҙ http://krespo.github.io/a.png мҷҖ http://krespo.github.io/b.png к°Җ м ҖмһҘлҗҳлҠ” кІҪмҡ°, к°Ғ м»¬лҹјмқҳ valueлҠ” a.pngмҷҖ b.pngл§Ң м ҖмһҘлҗңлӢӨ.

2. м••м¶•

Kuduм—җм„ңлҠ” мқёмҪ”л”© мқҙмҷём—җ м••м¶•мқ„ мқҙмҡ©н•ҳм—¬ л””мҠӨнҒ¬ мӮ¬мҡ©лҹү л°Ҹ scan мҶҚлҸ„лҘј нҡЁмңЁнҷ” н•  мҲҳ мһҲлӢӨ. м••м¶•л°©мӢқмқҖ LZ4, Snappy, zlibлҘј мӮ¬мҡ©н•  мҲҳ мһҲмңјл©°, мқјл°ҳм ҒмңјлЎң LZ4 м••м¶•мқҙ м••м¶• м„ұлҠҘмқҙ мўӢкё° л•Ңл¬ём—җ л§Һмқҙ мӮ¬мҡ©н•ңлӢӨ. zlib м••м¶•мқҖ к°ҖмһҘ мһ‘мқҖ лҚ°мқҙн„° мӮ¬мқҙмҰҲлЎң м••м¶•н•  мҲҳ мһҲкё°л•Ңл¬ём—җ л””мҠӨнҒ¬ мӮ¬мҡ©лҹүмқ„ мөңлҢҖлЎң мӨ„мқҙкі мһҗ н•ңлӢӨл©ҙ zlibлҘј мӮ¬мҡ©н•ҳл©ҙ лҗңлӢӨ.
